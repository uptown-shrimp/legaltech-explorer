Popularity Score

What the popularity score means

The popularity score is a 0–100 indicator designed to help you quickly understand how established, visible, and well-documented a legal tech tool is in the market.

It brings together multiple signals to provide a balanced, comparable view across tools.

How the score is calculated

Each tool's popularity score is based on a combination of:

Market interest

How often the tool is searched for over the past year.

Product maturity

Whether the tool is positioned as a quick-start, intermediate, or enterprise-grade solution.

AI capabilities

Whether the tool includes AI-powered functionality.

Geographic reach

The regions where the tool is available or actively used.

Data completeness

How much verified information is available about the tool.

Each factor contributes a weighted share to the overall score, which is calculated in advance and updated as new data becomes available.

Handling missing information

Not all tools have complete data across every category.

When information is unavailable, the score is calculated using the data that is available, rather than penalising the tool. This ensures newer or emerging tools can still be fairly compared.

Transparency

You can view a breakdown of the factors contributing to a tool's popularity score by opening its details. This is provided to help you understand why a tool scores the way it does.

How to Use the Legal-Tech Explorer

The Legal-Tech Explorer helps in-house legal teams discover and compare legal technology tools available in Australia.

1. Getting Started

There are two main ways to find the tools you need:

Search using natural language

Describe what you're looking for in plain English (for example: "AI contract automation for in-house teams").

Your search automatically applies relevant filters, which you can view and adjust manually at any time.

Browse all tools

Select View All Tools to explore the full database without applying filters.

The total number of tools available is displayed at the top of the page.

2. Browsing Tools

Tools are displayed in a grid layout, allowing you to scan and explore available options quickly.

Click on any tool to view its details

Results update automatically as filters are applied or adjusted

3. Filtering Tools

Use the Filters panel to refine your results.

A wide range of filters is available, covering areas such as:

AI capability

Product maturity

Adoption level

Practice areas

Regions served

Features and capabilities

Additional filter options can be found under the More Filters section.

Filters update in real time and remain active as you browse.

4. Comparing Tools

Adding tools to compare

Select tools from the results list to add them to comparison mode.

Comparison view

In comparison mode, selected tools are displayed side by side so you can easily compare key attributes.

AI comparison summary

Within the comparison view, you can generate an AI-powered summary that:

Compares tools against one another

Highlights key differences

Outlines strengths and limitations

Provides high-level suggestions to support decision-making

Edit Selection

Select 1–5 tools for your report
